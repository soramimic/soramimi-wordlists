# ADR 00008: plant(植物)ワードリストの新規作成と自動更新

- Status: accepted
- Date: 2026-07-11
- Related: 00007(sekitsui の自動更新。本リストはその設計を植物に踏襲)

## Context

動物(脊椎動物)の網羅リスト sekitsui(ADR 00007)に続き、植物の和名リストを新設する。sekitsui と同じく `surface = pronunciation = カタカナ和名` の構造なので、人名リスト(ADR 00005)で必要だった「記事冒頭文からの読み抽出」は不要で、Wikidata の日本語ラベルがそのまま答えになる。

植物は脊椎動物(約7,280種)と規模の桁が違う。被子植物だけで数十万種あり、Wikidata の SPARQL で被子植物 Q25314 を一括クエリすると WDQS がタイムアウトする。一方、カタカナ和名を**持つ**種に絞ると規模は現実的になる。事前調査(2026-07-11 時点)での実測は次の通り:

- 被子植物: 60 目に分割して取得し計 **6,025 種**(TIMEOUT なし)。上位はキク目540・イネ目499・シソ目448・キジカクシ目428・バラ目396
- 裸子植物 99、シダ植物 216(Pteridophyta 215 + Lycophyta 1)、コケ植物 101(蘚類62・苔類37・ツノゴケ類2)、藻類 114(紅藻38・褐藻41・緑藻23・車軸藻7・珪藻5)
- 合計 **約6,500 種**。sekitsui(7,280)に匹敵する網羅リストになる

## Decision

Wikidata を出典に `tools/update_plant.py` の年次バッチで未収録の種を追記する。ADR 00001 の4基準(機械可読な構造化データ / 認証不要 / CC0 / 差分追記)を満たし、sekitsui と同じ差分方式(既存行の表記・id は書き換えず、未収録の和名だけ id 連番で追記)を採る。

- 対象は **rank=種(Q7432)** で **日本語ラベルがカタカナのみ**の taxon。カタカナ以外(漢字ラベル、曖昧さ回避付き)は既存リストの性格に合わないので除外する
- **`class` 列に大分類(双子葉 / 単子葉 / 裸子植物 / シダ植物 / コケ植物 / 藻類)を持たせる**。被子植物は数が支配的なので、利用側のバランス取りのため双子葉/単子葉に細分する。双子葉は現代分類では多系統だが、伝統的な2分類として運用する
- **被子植物は目(order)ごとに分割クエリする**。被子植物 Q25314 の一括クエリはタイムアウトするため。目の一覧は実行時に Wikidata から取得(`?o wdt:P171* wd:Q25314 ; wdt:P105 wd:Q36602`)し、単子葉植物 Q78961 配下の目を「単子葉」、それ以外を「双子葉」に振り分ける。目単位ならタイムアウトしないことを実測で確認済み(最大のキク目540種でも可)。目一覧を実行時取得にすることで、APG の目の増減に追随できる
- **非被子植物は正式な門・綱の QID を直接指定**して取得する。コケ植物 Q29993・藻類 Q37868 は Wikidata 上で非公式グループであり P171 の親タクソンにならない(配下の種が 0 件になる)ため、門ごとに分けて束ねる:
  - 裸子植物 Q133712
  - シダ植物 = Pteridophyta Q178249 + ヒカゲノカズラ植物 Lycophyta Q157819
  - コケ植物 = 蘚類 Q25347 + 苔類 Q189808 + ツノゴケ類 Q191156
  - 藻類 = 紅藻 Q103169 + 緑藻 Q264543 + 褐藻 Q184573 + 珪藻 Q9642991 + 車軸藻 Q133219
- **化石種・絶滅種も含める**。判定は sekitsui と同じで、IUCN ステータス(P141)が絶滅種(Q237350)/野生絶滅(Q239509)、または instance of(P31)が化石タクソン(Q23038290)のいずれか。`extinct` 列(yes/no)で区別する
- 取得総数が下限(4,000)を下回ったら取得失敗とみなして中断する(実測 約6,500 に対する妥当性ガード)

## Consequences

- plant が pokemon / nations / stations / baseball / football / physicist / sekitsui に続く自動更新対象になる
- 藻類は多系統で、上記5門で網羅しきれない(渦鞭毛藻・ハプト藻など単細胞群の一部は漏れる)。網羅リストへの追記運用なので、取りこぼしは将来 `CLADES` に門を足して補える
- 双子葉/単子葉の判定は目の所属に依存する。Wikidata で単子葉植物 Q78991 配下に置かれる目(サトイモ目・ホンゴウソウ目を含む)を単子葉とする。将来 APG の再編で目の所属が変われば追随する
- 被子植物の目の QID・所属が変わってタイムアウトや件数急減が起きた場合は、妥当性ガード(下限4,000)で検出される。特定の目が将来巨大化してタイムアウトするなら、その目を科(family)単位に細分する
- 追記のみで、既存の和名の削除・改名の反映はしない(網羅リストへの追記として運用)
