# ADR 00010: 架空人物リストの画像を「カード」から「肖像のみ」に変更

- Status: accepted
- Date: 2026-07-13
- Supersedes: [00006](00006-fictional-scientist-wordlist.md) の Decision 2 のみ
- Superseded by: none

## Context

ADR 00006 では、架空科学者の画像に名前・生没年・業績などのテキストを生成時点で合成した
「カード」画像(1880x800)を配布し、soramimic-video 側の画像利用フロー(URLを1枚絵として
表示するだけ)を変えない方針とした。架空アニメキャラ2リストも同じカード形式を踏襲した。

その後 soramimic-video 側にレイアウト機能(JSONで画像とテキスト要素を宣言し、CSVの任意列を
テンプレート参照してフレームに合成する仕組み)が入り、テキストはレイアウト側で重ねられる
ようになった。テキスト焼き込み済みのカード画像には次の制約がある:

- レイアウトを変えたい場合(肖像だけを大きく出す、説明の文言や位置を変える等)に画像を
  再生成・再リリースする必要がある
- テキストが画像解像度(横1880px)でラスタライズ済みのため、動画解像度によっては
  レイアウト側で描くテキストよりぼやける
- 肖像部分だけを他の用途に使い回せない

## Decision

1. **配布画像は肖像のみ(720x720 JPEG)とし、テキストは合成しない。** 名前・説明等の表示は
   soramimic-video のレイアウトプリセット(`fictional_scientist_card` / `fictional_anime_card`)が
   CSVの列を参照して描画する(従来のカードと同等の見た目を再現するプリセットを用意)。
2. **肖像のみ画像は新しいリリースタグ(`fictional-scientist-v2` /
   `fictional-anime-character-v2` / `fictional-daily-anime-character-v2`)で配布し、
   CSVの `image` / `image_page` を新タグに差し替える。** アセットのファイル名
   (`fs_NNNN.jpg` 等)は v1 と同一に保ち、ID対応を維持する。
3. **v1(カード版)のリリースは削除せず残す。** テキスト焼き込み済みの1枚絵が必要な用途は
   引き続き v1 を参照できる。
4. 肖像のみ画像は v1 カードの肖像領域(40,40〜760,760)を切り出して作成する。元の肖像
   ファイルからの再書き出しではなくカードからの切り出しとすることで、公開済みIDとの
   1対1対応を機械的に保証する。

ADR 00006 のその他の決定(GitHub Release配布、列構成、AI生成の明示)は変更しない。

## Consequences

- レイアウト(文言・配置・配色)の変更は soramimic-video 側のレイアウトJSONの編集だけで
  済み、画像の再リリースが不要になる
- テキストは動画解像度で描画されるため、カード焼き込みよりくっきり表示される
- soramimic-video 以外でCSVの `image` を「説明付きの1枚絵」として使っていた利用側が
  あれば、v1 タグのURLに切り替えるか、テキスト表示を自前で用意する必要がある
- 今後 `fictional-scientists` 側から再エクスポートする場合は、肖像のみ画像のタグ
  (v2系)を `--url-prefix` に指定すること
